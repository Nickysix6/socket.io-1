Socket.io example chat

This is a project for socket.io chat example running on Heroku at https://fathomless-beyond-58796.herokuapp.com/

How to build and deploy on Heroku:

  1. Use Git to clone this project:
    git clone https://github.com/datjandra/socket.io.git

  2. Download and install Node.js

  3. Download and install Heroku Toolbelt

  4. Build the project locally in socket.io directory:
    npm install

  5. Start the process locally:
    heroku local

  6. Go to http://localhost:3000 and verify that chat server is running.

  7. If there are no errors then create a new instance on Heroku:
    heroku create

  8. Push code to heroku:
    git push heroku master
